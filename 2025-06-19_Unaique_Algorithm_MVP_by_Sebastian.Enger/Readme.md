## 🌟 Core Philosophy

The UNAIQUE Algorithm utilizes the **Formula of Duality Consciousness:** $N = \{x.n, y.m\}$. 
* **$x.n$ (Physical Aspect):** Represents the Yin, the physical data points, resilience, environment, and somatic reality.
* **$y.m$ (Mental/Spiritual Aspect):** Represents the Yang, the mental data points, cognitive style, intellect, and spiritual blueprint.

By extracting these data points from simple keyword inputs and metadata, the algorithm stimulates the reader precisely where they are, ensuring deep comprehension and emotional resonance.

## 🚀 Features

* **Deep Persona Profiling (`analyze_persona`):** Generates an exhaustive JSON profile of a user based on minimal input. The profile maps out:
    * Cognitive Load Preferences & Linguistic Intelligence
    * Psychological Profiles & Shadow Integration
    * Energetic Blueprints & Consciousness Frameworks (CSI-Enhanced)
* **Hyper-Tailored Content Generation (`write_persona_text`):** Writes or rewrites articles, ensuring the target persona intuitively grasps at least 80% of the concepts in their own language style, intellect, and emotional state.
* **Bulletproof JSON Parsing:** Features highly resilient JSON loaders (`safe_json_loads`) that can handle malformed AI outputs, markdown wrappers, and encoding issues seamlessly.
* **Secure File Operations:** Built for production environments with rigorous file permission checks, size limits, and directory traversal protections.

## 🏗️ Architecture

The MVP demonstrates two primary implementations:

1.  **ArtikelSchreiber.com Approach (Creation):** Takes main/sub-keywords and user metadata (like IP location) to dynamically build a new persona, and then generates an original, SEO-optimized article tailored exclusively for that profile.
2.  **Unaique.de Approach (Transformation):** Takes an existing body of text and rewrites it to match a predefined persona archetype (e.g., *casual*, *academic*, *scientific*), acting as a powerful communication bridge.
